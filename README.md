# cyptobox
Cyptobox is a Python library for dealing with cryptography algorithms.
## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install dependecies.

```bash
pip3 install pysqlite3
pip3 install pycryptodome
pip3 install rsa
pip3 install pyDes
```

## Usage

```python
python3 start.py
```
